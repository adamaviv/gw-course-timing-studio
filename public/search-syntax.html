Search Syntax

GW Course Studio supports a regex-like search language for course-number filtering. Plain text search still works for title, instructor, section, status, and meeting pattern.

Core Patterns

Pattern Meaning Example
6* Course numbers in the 6xxx band. Graduate-level classes.
62* Course numbers with prefix 62xx. 6200 through 6299.
1*-4* Inclusive wildcard range from 1xxx to 4xxx. Undergraduate classes.
62+ Course number greater than or equal to 6200. 6200 and above.
<3* / <=3* Level-aware comparator on wildcard bands. <3* means below 3000, <=3* includes 3xxx.
>6* / >=6* Level-aware comparator on wildcard bands. >6* means 7000+, >=6* includes 6xxx.

Boolean Join

Use || for OR. Spaces are AND within each side. Example: 62* ai || 8* means (62* AND ai) OR 8*.

Course Number Suffixes

Letter suffixes after 4-digit course numbers are ignored for numeric matching. Example: 2401W is treated as 2401.

If a token is not recognized as DSL, it is treated as normal text search.
